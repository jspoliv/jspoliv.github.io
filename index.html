Click on two pixels on the map to define a starting point and an end point for the pathfinding. The starting and end points cannot be black (walls/impassable terrain).

The map has NxN pixels. Each color has it's own pathing cost (bluest = lowest; green = mid; reddest = highest; black/wall = impassable terrain).

Starting point: (-1,-1)
Ending point: (-1,-1)

Pathing cost: -1
Time Elapsed: -1
